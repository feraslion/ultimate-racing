# Ultimate Racing 🏎️

مشروع لعبة سباقات متكاملة مبني باستخدام Unity 6.

## مميزات المشروع
- **نظام القيادة:** تحكم كامل في السيارة مع نظام نيترو.
- **الذكاء الاصطناعي:** سيارات منافسة تتبع مسارات محددة.
- **نظام الاقتصاد:** جمع العملات وشراء التطويرات.
- **نظام الحفظ:** حفظ التقدم تلقائياً باستخدام PlayerPrefs.
- **بناء تلقائي:** مجهز بـ GitHub Actions لبناء ملفات APK و EXE.

## هيكل المجلدات
- `Assets/Scripts/Core`: الأكواد الأساسية للمحرك.
- `Assets/Scripts/Racing`: أنظمة السباق واللفات.
- `Assets/Scripts/AI`: ذكاء المنافسين.
- `Assets/Scripts/UI`: واجهات المستخدم.

## كيفية التثبيت
1. قم بتحميل المشروع من GitHub.
2. افتح المجلد باستخدام Unity 6 LTS.
3. لبناء اللعبة، توجه إلى `File > Build Settings`.

## البناء التلقائي (CI/CD)
المشروع مجهز بملفات `GitHub Actions` التي تقوم ببناء اللعبة تلقائياً عند كل Push للمستودع.
*ملاحظة: تحتاج إلى إضافة أسرار Unity (License, Email, Password) في إعدادات المستودع لتفعيل البناء التلقائي.*
